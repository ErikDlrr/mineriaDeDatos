# Minería de Datos y Machine Learning

## Descripción del Proyecto
Este repositorio contiene implementaciones prácticas y proyectos analíticos enfocados en la extracción de conocimiento y la construcción de modelos predictivos. El objetivo principal del código, centralizado en el notebook `MineriaDeDatos2.ipynb`, es aplicar técnicas de Minería de Datos para explorar patrones, evaluar métricas de error y optimizar algoritmos en escenarios reales.

## Tecnologías y Herramientas
* **Lenguaje principal:** Python
* **Librerías utilizadas:** Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn
* **Entorno de desarrollo:** Jupyter Notebook / Google Colab

## Competencias Técnicas Demostradas
A lo largo de los ejercicios de este repositorio, se aplican los siguientes conceptos:
* **Aprendizaje Supervisado:** Entrenamiento, evaluación y ajuste de hiperparámetros de modelos como Random Forest y Gradient Boosting.
* **Aprendizaje No Supervisado:** Segmentación y análisis exploratorio de datos utilizando el algoritmo K-Means.
* **Resolución de Problemas:** Modelado matemático y optimización de lógica algorítmica aplicada a casos de estudio (por ejemplo, el "Problema del granjero").
